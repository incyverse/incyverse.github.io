❖ 로봇 관리 시스템 유지보수 및 고도화 (2023.07 ~ 2024.10)
◉ 역할 : 기획, 풀스택 개발
◉ 주요 업무
• AMR(Autonomous Mobile Robot)의 ROS(Robot Operating System)와 데이터 연동 및 웹 관제를 위한 REST API 개발
• ROS와 WebSocket 통신을 통해 실시간 데이터 수집
• 웹에서 AMR을 실시간으로 조작하고 데이터를 모니터링할 수 있는 시스템 아키텍처 설계
• 데이터베이스 구조 설계 및 시스템 성능 최적화
• JWT(JSON Web Token) 기반 로그인 시스템 개발
• SSE(Server-Sent Events)를 통한 메시지 처리 개발
◉ 성과
• AMR 데이터를 실시간으로 모니터링하고 제어할 수 있는 안정적인 REST API 설계 및 개발로 시스템 성능 향상
• NestJS와 TypeORM을 기반으로 한 성능 최적화로 API 응답 속도 개선
• Docker를 활용한 컨테이너화로 시스템 배포와 유지보수의 효율성 증대
◉ 상세 내용
• AMR에서 실행되는 ROS와 연동하여 웹에서 실시간 모니터링이 가능하도록 WebSocket 기반의 PUB/SUB을 개발
• ROS와 연동으로 수집된 데이터로 사용자는 AMR의 상태 정보와 작업 현황을 실시간으로 확인하고 원격으로 AMR을 제어
• ROS와 WebSocket을 사용해 AMR에서 발생하는 데이터를 실시간으로 저장하고 이를 사용자에게 시각화하여 제공
• WebSocket을 통해 양방향 데이터 전송이 가능해 AMR이 수행하는 작업의 상태를 실시간으로 수집하고 명령을 전달할 수 있는 기능 구현
• AMR의 데이터를 실시간으로 모니터링하고 제어할 수 있는 웹 기반 시스템 아키텍처 설계
• 데이터 처리, 실시간 알림 및 메시지 전송, 실시간 제어 기능을 구현
• 데이터베이스 구조 설계와 성능 최적화를 통해 고성능 실시간 처리 구현
• 시스템에서 필요한 대량의 데이터를 효율적으로 처리하고 성능을 높이기 위해 데이터베이스 구조를 설계 및 최적화
• AMR의 작업 로그와 실시간 데이터를 저장하고 관리
• 데이터를 효율적으로 저장하고, 빠른 조회와 처리가 가능하도록 데이터베이스 구조 설계
• 시스템 전체 성능을 최적화하기 위해 데이터 캐싱 전략과 데이터베이스 인덱싱을 도입하여 데이터 처리 속도 개선
• 시스템 보안을 강화하기 위해 JWT 기반으로 사용자 인증 시스템을 도입하여 보안성 강화
• 사용자 세션 관리의 복잡성을 줄이고 무결성과 보안을 확보
• 서버에서 클라이언트로 데이터를 지속적으로 보내기 위해 SSE를 활용하여 시스템 내에서 발생하는 이벤트를 사용자에게 즉시 전달하여 사용자는 실시간으로 AMR의 상태 변화를 모니터링 가능
~~• 실시간 데이터를 안정적으로 처리할 수 있는 REST API를 통해 AMR 데이터를 실시간으로 모니터링하고 제어할 수 있는 기능을 개발함으로써 시스템의 성능을 크게 향상시켰습니다. 이를 통해 AMR 데이터를 빠르고 정확하게 모니터링하고 제어할 수 있었습니다.~~
• NestJS와 TypeORM을 기반으로 API 성능을 최적화하여 응답 속도를 개선
• 시스템 복잡성을 줄이고 API 호출 횟수를 최소화하여 전반적인 성능 향상
• Docker를 활용하여 시스템의 배포 및 유지보수를 효율적으로 처리
~~• 개발 환경의 일관성을 유지하고 배포 과정에서 발생하는 문제 감소~~
◉ 기술 스택 : NestJS, TypeORM, JWT, TypeScript, React, Redux, Docker, MariaDB, SCSS, SQLite, Material-UI, PM2, GitHub, Yarn
